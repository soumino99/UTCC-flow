# UTCC-flow

UTCC新規入会者向けの、Yes/Noで答えるコーヒー活動診断です。  
`index.html` をブラウザで開くと、回答内容に応じて質問順が変わる（アキネーター風）診断でおすすめの参加先を提案します。

## 実装内容

- 質問形式: 全問 `Yes / No`
- 質問数: 最大12問（確信度が高まると早めに結果確定）
- 質問画面に進捗バー表示あり
- 質問画面に質問数表示あり（`x / 12問`）
- 判定方式: 動的出題 + スコアリング方式
- 結果画面: 他の結果へ直接ジャンプ可能

## 質問一覧（改善版）

1. コーヒーそのものは好き？
2. 初対面の人とも話せる場が好き？
3. コーヒーの理論を体系的に学びたい？
4. 味や香りの違いを言語化して共有したい？
5. 家で淹れる習慣を作りたい？
6. ハンドドリップの再現性を上げたい？
7. エスプレッソマシンに触れてみたい？
8. ラテアートを本気で練習したい？
9. 焙煎を最初から最後までやってみたい？
10. 新しいカフェを開拓するのが好き？
11. 競技会レベルのこだわりにも興味がある？
12. イベントを運営側で作る役割に興味がある？

## 結果カテゴリ

- 定例会
- カフェ巡り
- 焙煎会
- エスプレッソ会
- セミナー
- 各種運営

## 分岐アルゴリズム（実装準拠）

- 6カテゴリ（定例会 / カフェ巡り / 焙煎会 / エスプレッソ会 / セミナー / 各種運営）に対してスコアを持つ
- 回答済み内容とカテゴリ別プロファイルの一致度でスコア更新（一致は加点、不一致は減点）
- 毎回答後に、現在の候補を最も絞り込める質問を次に出題（質問順が固定でない）
- 6問以上回答後、1位と2位のスコア差が一定以上なら早期に結果確定
- 未確定のままでも最大12問で必ず結果確定
- 最終的に最もスコアが高いカテゴリを結果として採用
- 同点時は以下の優先順で決定
  1. 各種運営
  2. 焙煎会
  3. エスプレッソ会
  4. カフェ巡り
  5. セミナー
  6. 定例会

## 実行方法

1. `index.html` をブラウザで開く
2. 「診断スタート」を押す
3. 質問に `Yes/No` で回答する（回答に応じて次の質問が変わる）
4. 結果画面でおすすめ活動を確認する
5. 結果画面の「他の結果へジャンプ」から別の結果も確認できる
