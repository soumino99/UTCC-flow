# UTCC-flow

UTCC新規入会者向けの、Yes/Noで答えるコーヒー活動診断です。  
`index.html` をブラウザで開くと、回答内容に応じて質問順が変わる（アキネーター風）診断でおすすめの参加先を提案します。

## 実装内容

- 質問形式: 全問 `Yes / No`
- 質問数: 最大24問（確信度が高まると早めに結果確定）
- 質問画面に進捗バー表示あり
- 質問画面に表情絵文字で確信度を表示（数値は非表示）
- 判定方式: 動的出題 + スコアリング方式
- 結果画面: 他の結果へ直接ジャンプ可能
- 特殊分岐: 全質問に `No` と回答した場合はパロディ結果を表示

## 結果カテゴリ

- 定例会
- カフェ巡り
- 焙煎会
- エスプレッソ会
- セミナー
- 各種運営

## 分岐アルゴリズム（実装準拠）

- 6カテゴリ（定例会 / カフェ巡り / 焙煎会 / エスプレッソ会 / セミナー / 各種運営）に対してスコアを持つ
- 回答済み内容とカテゴリ別プロファイルの一致度でスコア更新（一致は加点、不一致は減点）
- 毎回答後に、期待情報利得（エントロピー減少）が最大の質問を次に出題
- 追加で `Yes/No` がほぼ `1/2` に割れる質問を優先し、検索空間を効率的に半分へ圧縮
- 10問以上回答後、1位カテゴリの確率が高く、2位との差も十分あるときに早期確定
- 未確定のままでも最大24問で必ず結果確定
- 最終的に最もスコアが高いカテゴリを結果として採用
- 同点時は以下の優先順で決定
  1. 各種運営
  2. 焙煎会
  3. エスプレッソ会
  4. カフェ巡り
  5. セミナー
  6. 定例会

## 実行方法

1. `index.html` をブラウザで開く
2. 「診断スタート」を押す
3. 質問に `Yes/No` で回答する（回答に応じて次の質問が変わる）
4. 結果画面でおすすめ活動を確認する
5. 結果画面の「他の結果へジャンプ」から別の結果も確認できる
