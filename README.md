# UTCC-flow

UTCC新規入会者向けの、Yes/Noで答えるコーヒー活動診断です。  
`index.html` をブラウザで開くと、10個の質問に答えておすすめの参加先を提案します。

## 実装内容

- 質問形式: 全問 `Yes / No`
- 質問数: 10問
- 質問画面に進捗バー表示あり
- 質問画面に質問数表示あり（`x / 10問`）
- 判定方式: 10問回答後のスコアリング方式
- 結果画面: 他の結果へ直接ジャンプ可能

## 質問一覧（改善版）

1. コーヒーそのものは好き？
2. 苦味のあるブラックも楽しめる？
3. 豆・抽出・焙煎の知識を増やしたい？
4. 味や香りの違いを言語化して共有したい？
5. 家でコーヒーを淹れる習慣を作りたい？
6. ハンドドリップを継続して練習したい？
7. エスプレッソやラテを自分で作りたい？
8. 焙煎（生豆から）に取り組みたい？
9. 新しいカフェを開拓するのが好き？
10. 学祭のような場を運営側で作ってみたい？

## 結果カテゴリ

- 定例会
- カフェ巡り
- 焙煎会
- エスプレッソ会
- セミナー
- 各種運営

## 分岐アルゴリズム（実装準拠）

- 6カテゴリ（定例会 / カフェ巡り / 焙煎会 / エスプレッソ会 / セミナー / 各種運営）に対してスコアを持つ
- 各質問の `Yes/No` に応じて、カテゴリ別に加点
- 相関の強い組み合わせ（例: `Q3 & Q4`, `Q5 & Q6` など）に追加加点
- 最終的に最もスコアが高いカテゴリを結果として採用
- 同点時は以下の優先順で決定
  1. 各種運営
  2. 焙煎会
  3. エスプレッソ会
  4. カフェ巡り
  5. セミナー
  6. 定例会

## 実行方法

1. `index.html` をブラウザで開く
2. 「診断スタート」を押す
3. 10問に `Yes/No` で回答する
4. 結果画面でおすすめ活動を確認する
5. 結果画面の「他の結果へジャンプ」から別の結果も確認できる
